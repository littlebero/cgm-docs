## 사용 순서

1. **CGM_소개서** 읽기 → 에스컬레이션 블록 시스템 프롬프트에 삽입
2. **CGM_프로젝트_스냅샷_템플릿** 복사 → 현재 상태 채우기
3. `claude.ai` → CGM 프로젝트 열기 → 채운 템플릿 + 관련 파일 첨부 → 대화 시작
